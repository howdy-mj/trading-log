**Frontend**: React, Redux, redux-thunk, redux-toolkit, @emotion/styled, [is-validated](https://github.com/howdy-mj/is-validated), Storybook

**Backend**: Firebase(Auth, Realtime Database)

**Deploy**: Vercel
