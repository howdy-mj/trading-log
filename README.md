# 매매일지 '티록(T-Log)'

코인 거래 매매일지 기록 (<i>개발중</i>)

### 기술 스택

**Frontend**: React, Redux, redux-thunk, redux-toolkit, @emotion/styled, react-quill, [is-validated](https://github.com/howdy-mj/is-validated), Storybook

**Backend**: Firebase(Auth, Realtime Database)

**Deploy**: AWS Amplify
